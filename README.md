# Skeleton

It is a start point for any projects.

To start the project, follow these steps:
1) add favicon.svg file to *"frontend/favicon"*
3) run **"npm install"**
2) run **"npm run favicon"**
3) run **"npm run serve"**

It will run localhost web-server for html creation. It uses BrowserSync for update html in browsers. 

##Commands (npm run):
1) **"favicon"** - create favicon files and html for any devices
2) **"build"** - build html, js, css for development mode
3) **"serve"** - run localhost server and watcher
4) **"watch"** - run watcher for files. Create development mode assets
5) **"build-prod"** - create assets for production
6) **"new-block"** - create new block folder with base files in *"frontend/blocks/"*
